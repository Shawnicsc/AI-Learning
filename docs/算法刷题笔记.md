## 滑动窗口

### 定长滑动窗口

思想：枚举右 维护左

模板：

- 入：右入窗口 判断是否符合条件
- 更新：达到最大窗口长度，判断答案是否需要更新
- 出：左端点离开窗口，判断是否需要减少

```python
class Solution:
    def maxVowels(self, s: str, k: int) -> int:
        # 枚举右 维护左
        res = vowel = 0
        for i,c in enumerate(s):
            # in
            if c in "aeiou":
                vowel += 1
            left = i-k+1
            if left < 0:
                continue
            
            # update answer
            res = max(res, vowel)
            if res == k:
                break

            # out
            if s[left] in "aeiou":
                vowel -= 1
            
        return res
             
```
